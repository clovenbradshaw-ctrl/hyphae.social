‚ùì Troubleshooting
  • "Wrong username or password" - Make sure you're copying the complete username including :hyphae.social
  • "Can't find homeserver" - Enter exactly: hyphae.social
  • "Invalid homeserver" - Some clients need https://hyphae.social with the protocol
  • QR code login - Only works between already logged-in devices, use manual entry for first login
